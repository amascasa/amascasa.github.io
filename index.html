Service Workers & PWA

Progressive Web Applications

Progressive Web Applications (PWA) are a powerful set of tools to use when developing non-native applications. The PWA enviroment allows the designer of an app to produce native-like apllications. Part of the process is storing critial files that can be loaded in case of no internet connection. This allows the app to function regardless of a connection, making the app much more reliable. Another benefit is speed. With the critical files stored locally retrieving them is less cpu intensive. Also with four different options on setting the viewport a developer can achieve a native app appearence without the browser bar.
