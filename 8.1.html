The Mystery of AJAX

The wonders of AJAX is that the browser can receive updates and modify only those elements without reloading the whole page. After the page has loaded, data can still be requested and recieved and the data can move in the background creating a more streamlined experience for the user. To formulate a request the first part is to create a an XMLHttpsRequest object, which takes the appearance of:

const xhttp = new XMLHttpRequest()

Then an if-statement is created to check the status of the request using the '.readyState' and '.status' methods. If they return "4" and "200" respectively which indicates the server is ready then by using '.responseText' the data can be sent to the html page to be displayed. to retrieve the data. the function sends the request to the server, then receives the response, and then processes the data through javaScript. Wonderful stuff!
