Explanation AJAX

I don't really know what AJAX is. I suppose it is used to retrieve information from the server, whatever that is.

The beauty of AJAX is that the browser can recieve updates and modify only those elements without reloading the whole page. To formulate a request the first part is to create a function that requests data from a web server. This is done first by creating a new browser built-in XMLHttpRequest object, which looks like this: const xhttp = new XMLHttpRequest() Once the object is created it can be used on an onload event function to retrieve the data. the function sends the request to the server, then receives the response, and then processes the data through javaScript.
