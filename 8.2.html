We have Two API's here that I believe work wonderfully together, with any web page. Random dog photos coupled with random cat facts with API's Randomdog and Cat Facts.
First diplayed you will find the random dog photos, which are always delightful. A slight drawback with this API; it doesn't always return a delightful dog photo. But, I'd say about 90% of the time and 90% ain't bad.
This API returns a JSON file that contains two properties: 'message', which contains the image url, and 'status' which returns success or fail. The devolper has generously supplied this API at no cost but accepts donations from appreciative users.
Accompanning these delightful photos are wonderful random cat facts. This API also has the drawback of also not returning cat facts every time, and in this case it's closer to a 60% success rate. Nonetheless, still a perfect addition to dog photos as it still adds to the random flare. Also the term 'facts' should be regarded very loosly but again, just as entertaining.
